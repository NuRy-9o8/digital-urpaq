## ⚠️ Решение проблем

### ❌ Ошибка: "Node.js не найден"
**Решение**: Установите [Node.js 18+](https://nodejs.org/)

### ❌ Ошибка: "Port 5000 уже используется"
**Решение**: 
```powershell
# Найдите процесс
netstat -ano | findstr :5000

# Если процесс 12345, убейте его
taskkill /PID 12345 /F
```

### ❌ Ошибка: "Port 3000 уже используется"
**Решение**:
```powershell
netstat -ano | findstr :3000
taskkill /PID <PID> /F
```

### ❌ Frontend не подключается к Backend
**Решение**:
- Убедитесь, что оба (Backend и Frontend) запущены
- Проверьте консоль браузера (F12 → Console)
- Перезагрузите страницу (Ctrl+R)

### ❌ База данных повреждена
**Решение**:
1. Остановите Backend (Ctrl+C)
2. Удалите файл: `backend\database.db`
3. Перезапустите Backend
4. БД пересоздастся с демо-данными

## Telegram bot quick start

Бот проекта: `@du_notifications_bot`.

1. Откройте файл `backend\.env`.
2. Укажите токен бота:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_из_BotFather
TELEGRAM_POLLING=true
```

3. Запустите backend:

```powershell
cd backend
npm start
```

4. Откройте Telegram, найдите `@du_notifications_bot` и нажмите `/start`.

Для локальной разработки без запуска Telegram polling можно временно выполнить:

```powershell
$env:TELEGRAM_POLLING="false"
npm start
```
